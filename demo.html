Key Features to Test

Farmer Features

  • Add new products with photos
  • Set wholesale/retail pricing
  • Grade products (A/B)
  • Manage customer orders
  • Schedule deliveries
  • View earnings dashboard

Consumer Features

  • Browse and filter products
  • Add items to cart
  • Save favorite products
  • Easy checkout process
  • Location-based matching
  • Track order history

Wholesaler Features

  • Bulk ordering (100kg+)
  • Partner with farms
  • Schedule farm visits
  • Wholesale pricing
  • Flexible delivery terms
  • Direct farmer contact

How to Start

  1. Go to Homepage: Click "Start Demo" or visit index.html
  2. Register/Login: Use any demo account from above
  3. Explore Dashboard: Each user type has a different interface
  4. Test Features: Try adding products, placing orders, etc.
  5. Switch Users: Logout and try different account types
